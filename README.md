# 🏥 ClinicFlow

> Enterprise Clinic Management System built using **.NET 10**, **PostgreSQL**, **Clean Architecture**, and designed to be **Azure-ready**.

ClinicFlow is a production-style healthcare management platform being built to master enterprise software development using modern .NET technologies.

This project follows real-world software engineering practices used in enterprise applications.
